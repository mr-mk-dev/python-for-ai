> **A structured, self-paced practice repository covering Python fundamentals to Machine Learning — 416 questions across 20 phases.**

This repository is my personal journey through Python, built from the ground up with the goal of mastering the language for **AI & Machine Learning**. Each phase focuses on a core concept, starting from basic I/O and progressing all the way to mini ML projects.

## 🎯 About

This project follows a **20-phase roadmap** designed to build Python skills from scratch, specifically tailored for someone aiming to get into **Machine Learning and Data Science**. Each phase contains hands-on practice questions with working solutions.

**What makes this different?**
- 🧱 **Structured learning** — each phase builds on the previous one
- 💻 **Hands-on code** — every question has a working Python solution
- 🎯 **ML-focused** — the roadmap is designed to prepare for NumPy, Pandas, and ML
- 📈 **Progressive difficulty** — from `print("Hello World")` to building ML models

## 🚀 How to Run

**Prerequisites:**
- Python 3.x installed ([Download Python](https://www.python.org/downloads/))

**Run any phase:**
```bash
# Navigate to the project directory
cd PythonProject

# Run a specific phase
python "PHASE 1 - INPUT & OUTPUT/Question.py"
python "PHASE 2 - VARIABLES & OPERATORS/Question.py"
python "PHASE 3 - CONDITIONS/Question.py"
python "PHASE 4 - LOOPS/Question.py"
```

> **Note:** Some solutions in Phase 3 and 4 are commented out (since they use `input()`). Uncomment the specific question you want to run.
